# Veti‑Vision

Elite veterinary command center built with:

- Vite + React + TypeScript
- Tailwind CSS + shadcn‑ui
- Framer Motion

## Getting Started

```sh
npm install
npm run dev
```

## Scripts

- dev: start the development server
- build: production build
- preview: preview the production build

## Notes

- Workflow, RBAC, and Audit Trails are implemented with lightweight local persistence for easy prototyping. Replace with your API when ready.
